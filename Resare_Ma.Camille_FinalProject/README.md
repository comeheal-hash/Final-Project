# CLI Student Record Management System

## Project Description
The CLI Student Record Management System is a Command Line Interface (CLI) application developed using Python. This system allows users to manage student records by adding, viewing, searching, updating, deleting, and sorting student information. The application demonstrates the use of Object-Oriented Programming (OOP), file handling, data structures, algorithms, and error handling.

---

## Features
- Login System
- Add Student Record
- View Student Records
- Search Student by ID
- Update Student Information
- Delete Student Record
- Sort Students by Average Grade
- JSON File Storage
- Error Handling and Input Validation
